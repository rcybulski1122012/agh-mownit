Aby uruchomić program należy:
- [Opcjonalne] Utworzyć i aktywować wirtualne środowisko
  `python3 -m venv venv && source ./venv/bin/activate`
- Zainstalować potrzebne pakiety
  `pip install -r requirements.txt`
- Uruchomić jupiter notebook'a
  `jupyter notebook`
- Po otwarciu przeglądarki wybrać plik `lab1.ipynb`

Drugim sposobem na uruchomienie programo jest skorzystanie z jego konsolowej wersji.
Dwa pierwsze kroki są identyczne jak w poprzednim przykładnie.
`python -m lab1`
